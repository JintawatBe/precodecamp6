<hr> & <br> เป็นการขึ้นบรรทัดใหม่เหมือนกัน hr จะเป็นการขึ้นบรรทัดใหม่แบบมีเส้นใต้ขีด br จะเป็นการขึ้นบรรทัดใหม่แบบไม่มีเส้น
<ul> & <ol> เป็นการเรียงลำดับเหมือนกัน ul เป็นการเรียงลำดับแบบสัญลักษณ์ ol เป็นการเรียงลำดับแบบตัวเลข
<select> & <checkbox> & <radio button> เป็นตัวเลือกในหน้าฟอร์มเหมือนกัน select เลือกได้หลายรายการ และแสดงผลรายการที่เลือก checkbox เป็นช่องเลือกที่สามารถเลือกได้หลายรายการ radio button เป็นรูปแบบการเลือกได้เพียงรายการเดียว
<div> & <span> เป็นกล่องข้อความเหมือนกัน div เมื่อจบข้อความจะขึ้นบรรทัดใหม่ span ข้อความสามารถต่อในบรรทัดเดียวกันได้
<script> & <style> เป็น element เหมือนกัน script ถูกใช้กำหนดคำสั่งของ java script style ถูกใช้กำหนดคำสั่งของ css
